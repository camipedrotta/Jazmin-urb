# Jazmín Urbano — sitio estático

Estructura: `index.html`, `styles.css` y carpeta `img/` con todas las imágenes.

## Publicar en GitHub Pages
1) Subí todo el contenido de `jazmin-urbano-site/` a un repositorio.
2) En **Settings → Pages**, elegí **Deploy from a branch**, Branch: `main`, carpeta `/root`.
3) Guardá. El link aparece abajo en la misma página.

> Si alguna imagen no carga por acentos/espacios, renombrala y actualizá la ruta en el HTML/CSS. Ya incluimos duplicados de nombres para el logo y la foto de suscripción.
